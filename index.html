About

This website was built by Daniel Corcoran to serve as an enduring portfolio piece. Daniel Corcoran is a self taught data specialist, who has worked as a data analyst and data engineer for the state government.

Daniel's main strengths are building solutions using data and working hands on with data, this includes; building websites, visualizing data, analysis, data engineering and machine learning. He achieves these through tools such as python, tableau, git and excel.

Daniel believes in continuous learning, good version control, clean code and delivering solutions of high quality. Feel free to check out his projects below...
